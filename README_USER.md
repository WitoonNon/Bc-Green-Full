# คู่มือการใช้งานระบบ BC Green (User Manual)

ระบบบริหารจัดการงานซ่อมและบริการแบบครบวงจร สำหรับผู้ใช้งานทั่วไปและเจ้าหน้าที่

---

## 📱 สำหรับผู้ใช้งานทั่วไป (Customers)

### 1. การแจ้งซ่อม (Repair Request)
- ไปที่เมนู **"แจ้งซ่อม"**
- กรอกรายละเอียดปัญหา, เลือกหมวดหมู่, และระบุรุ่นอุปกรณ์/หมายเลขซีเรียล
- สามารถแนบรูปภาพประกอบความเสียหายได้
- เมื่อบันทึกเสร็จ จะได้รับ **รหัสติดตามงาน (Tracking Code)** สำหรับเช็คสถานะ

### 2. การติดตามสถานะ (Track Status)
- ไปที่เมนู **"สถานะงาน"**
- ใส่รหัส Tracking Code ที่ได้รับ เพื่อดูความคืบหน้าของงานซ่อม
- ระบบจะแสดง Timeline ว่าตอนนี้อยู่ในขั้นตอนใด (เช่น กำลังตรวจเช็ค, กำลังซ่อม, หรือซ่อมเสร็จแล้ว)

### 3. ศูนย์ช่วยเหลือและความช่วยเหลือ (Support Center)
- **คู่มือการใช้งาน (Manuals):** ค้นหาและอ่านคู่มือการใช้งานอุปกรณ์เบื้องต้น
- **คำถามที่พบบ่อย (FAQs):** ค้นหาวิธีแก้ไขปัญหาเบื้องต้นด้วยตัวเอง

### 4. บทความและประกาศ (News & Promotions)
- ติดตามข่าวสารประชาสัมพันธ์และโปรโมชันล่าสุดได้ที่หน้าแรก

---

## 🛠️ สำหรับเจ้าหน้าที่ (Staff / Technicians)

เจ้าหน้าที่สามารถเข้าถึงระบบงานซ่อมโดยตรงเพื่ออัปเดตสถานะให้ลูกค้า

- **รับงานซ่อม:** ดูรายการแจ้งซ่อมใหม่ที่เข้ามาในระบบ
- **อัปเดตสถานะ:** ปรับเปลี่ยนสถานะงาน (เช่น ตรวจสอบแล้ว, กำลังซ่อม) พร้อมใส่หมายเหตุเพื่อให้ลูกค้าเห็นในหน้าติดตามสถานะ
- **จัดการคิวงาน:** ดูภาพรวมงานซ่อมทั้งหมดที่ตนเองรับผิดชอบ

---

## 👑 สำหรับผู้ดูแลระบบ (Admin)

ผู้ดูแลระบบสามารถเข้าใช้งานหน้า **Backoffice** ได้เต็มรูปแบบเพื่อควบคุมทั้งระบบ

### 1. การจัดการผู้ใช้และสิทธิ์ (User & Role Management)
- **ตรวจสอบผู้ใช้:** ดูรายชื่ออีเมลและชื่อของผู้ลงทะเบียนทั้งหมด
- **เปลี่ยนสิทธิ์ (Assign Roles):** สามารถปรับเปลี่ยนสิทธิ์ผู้ใช้ปกติให้เป็น `Technician` (ช่าง) หรือ `Admin` (ผู้ดูแล)
- **Sync ข้อมูล:** กรณีมีผู้ใช้ใหม่แต่รายชื่อยังไม่ปรากฏ สามารถใช้ปุ่ม **Sync Users** เพื่อดึงข้อมูลล่าสุดจากระบบ Authentication

### 2. การบริหารจัดการเนื้อหา (Content Management)
- **โปรโมชันและข่าวสาร:** เพิ่ม/ลบ/แก้ไข แบนเนอร์และประกาศที่จะแสดงที่หน้าแรก
- **ศูนย์ช่วยเหลือ (Support Center):** 
  - จัดการหมวดหมู่และเนื้อหาใน FAQ (คำถามที่พบบ่อย)
  - อัปโหลดไฟล์คู่มือ (User Manuals) ในรูปแบบ PDF หรือรูปภาพ
- **ข้อมูลทางเทคนิค:** จัดการข้อมูลรุ่นรถหรืออุปกรณ์มาตรฐานในระบบ

### 3. การตรวจสอบระบบ (System Monitor)
- ตรวจสอบสถานะการเชื่อมต่อ Database และระบบเก็บข้อมูลรูปภาพ (Storage)

---

## ❓ ปัญหาที่พบบ่อย (Troubleshooting)

- **เข้าหน้า Admin ไม่ได้:** บัญชีของคุณต้องมี Role เป็น `admin` หรือ `technician` ในฐานข้อมูลเท่านั้น
- **รูปภาพไม่โหลด/อัปโหลดไม่ได้:** 
  - ตรวจสอบว่าไฟล์ที่อัปโหลดมีขนาดไม่เกินที่กำหนด (แนะนำไม่เกิน 5MB)
  - หากระบบฟ้องเรื่อง Storage Error โปรดติดต่อฝ่ายเทคนิคเพื่อเช็คสถานะ Vercel Blob / Firebase Storage
- **หาชื่อผู้ใช้ไม่เจอ:** ให้แอดมินกดปุ่มรีเฟรชข้อมูลหรือใช้สคริปต์ Sync ข้อมูลเพื่อดึงรายชื่อล่าสุด
